# Planning

DM prep and reference material that doesn't belong in session notes. This folder holds long-running campaign structure, encounter design docs, and NPC reference guides.

## What Goes Here

- **Campaign arc outlines** — multi-session story arcs, major plot beats
- **Encounter tables** — d6/d12 tables of possible encounter scenarios for flexible prep
- **Reveal FAQs** — what happens when the party learns a major secret (one doc per reveal)
- **NPC roleplay guides** — quick reference sheets for portraying complex NPCs
- **Guest player briefings** — campaign-world introduction for one-shot guests
- **Session Notes Template** — the blank template used for each session

## Session Notes Template

`[[Session Notes Template]]` — copy this to `01 - session-notes/` and rename for each session.

## Encounter Design Philosophy (Lazy DM)

The session notes template uses the [Lazy Dungeon Master](https://slyflourish.com/returnofthelazydm.html) structure:

1. **Strong Start** — the first thing that happens; get everyone engaged immediately
2. **Scenes** — 3-7 potential scenes (not a linear path)
3. **Secrets & Clues** — 10 pieces of information the party might discover
4. **Fantastic Locations** — 2-3 vivid, evocative place descriptions
5. **NPCs** — brief notes on any NPCs appearing this session
6. **Monsters** — stat block references and DPR estimates
7. **Treasure** — specific items or gold if earned

## Reveal FAQ Format

For major plot reveals, create a dedicated FAQ file:

```markdown
# [Villain/Secret] — Reveal FAQ

## When does this get revealed?

## How do the players learn it?

## How does [NPC] react when confronted?

## What does this change about the party's understanding of the story?

## Follow-up hooks
```
