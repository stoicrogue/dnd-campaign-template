# Monsters

Creature stat blocks for encounters. Unlike adversaries, these are the mechanical/reference pages for creatures without major story roles.

## What Goes Here

- Standard monster stat blocks (SRD creatures you've homebrewed or annotated)
- Custom homebrew creatures
- Minion stat blocks (MCDM-style or custom minion rules)
- Reskinned monsters with renamed abilities

## What Does NOT Go Here

- Major villains with story roles → use `adversaries/` instead
- Allies who might enter combat → use `allies/` and link a stat block

## Stat Block Format

Use a consistent format for quick reference at the table:

```markdown
# [Monster Name]

**CR:** X | **HP:** X (XdX+X) | **AC:** X | **Speed:** X ft.

**STR** X (+X) | **DEX** X (+X) | **CON** X (+X)
**INT** X (+X) | **WIS** X (+X) | **CHA** X (+X)

**Skills:** ... | **Senses:** ... | **Languages:** ...

## Traits

## Actions

## Bonus Actions / Reactions / Legendary Actions
```
