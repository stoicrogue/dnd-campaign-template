# Allies

Friendly NPCs with narrative lore pages. These are characters the party has meaningful relationships with — recurring allies, quest givers, patrons, or contacts.

## Narrative Lore Page Format

Each ally gets a narrative page (not a stat block page). Focus on who they are, not their mechanics.

```markdown
# [NPC Name]

- **Role:** [Their function in the story]
- **Affiliation:** [[Faction Name]]
- **Location:** [[Home Base or Current Location]]
- **Status:** Active / Missing / Dead

## Background

## Personality & Motivations

## Relationship to the Party

## Quotes

> "[A memorable line from play]"

## DM Notes

*Private DM information — secrets, future plans, hidden agendas.*
```

## Stat Blocks (if needed)

If an ally might enter combat (escort missions, final battles), create a separate stat block file:
- `Ally Name.md` — narrative lore page (linked here)
- `Ally Name - Stat Block.md` — mechanical stats

Link from the narrative page with `[[Ally Name - Stat Block]]` under a `## Stat Block` heading.
