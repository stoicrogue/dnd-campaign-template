# Main Party

Player characters and DM reference sheets for the current party.

## Suggested Files

- **One file per PC** — name, class, background, personality, goals, secrets
- **Party Reference Sheet** — initiative modifiers, passive perception, languages, key abilities at a glance
- **Contact List** — important NPCs the party knows and their relationship level

## PC Page Format

```markdown
# [Character Name]

- **Player:** [Player Name]
- **Class/Level:** [Class] [Level]
- **Race:** [Race]
- **Background:** [Background]

## Personality

## Goals & Motivations

## Secrets & Backstory Hooks

## Notes from Sessions
```
