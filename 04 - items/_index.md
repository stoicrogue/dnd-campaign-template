# Items

Magic items and homebrew equipment. One file per notable item.

## Item Page Format

```markdown
# [Item Name]

- **Type:** Weapon / Armor / Wondrous Item / Consumable
- **Rarity:** Common / Uncommon / Rare / Very Rare / Legendary / Artifact
- **Attunement:** Required / Not required
- **Current Holder:** [[Character Name]]

## Description

*Appearance, feel, magical aura.*

## Properties

*Game mechanics — what it does.*

## History & Lore

*Where did it come from? Who made it? What is its significance?*

## DM Notes

*Future reveals, upgrades, or plot hooks tied to this item.*
```

## Item Progression

For items that grow with the party (sentient weapons, artifacts), track their progression:

```markdown
## Awakened Properties

*Additional abilities gained after [Session X] / [Quest completion].*
```
