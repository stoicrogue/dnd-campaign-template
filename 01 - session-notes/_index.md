# Session Notes

Per-session DM prep notes. One file per session.

## Naming Convention

```
[Campaign Name] Session [NUMBER] (YYYY-MM-DD).md
```

Example: `Tempus Session 23 (2024-03-15).md`

## Getting Started

Copy the session prep template from `[[05 - planning/Session Notes Template]]` and rename it for each session.

Run `add_yaml_frontmatter.py` (from this directory) to bulk-add frontmatter to existing session files:
```bash
python ../add_yaml_frontmatter.py
```

## Tips

- Write notes in **anticipatory** tense during prep ("The party will encounter...")
- After the session, update to **past tense** and note what actually happened vs. what was planned
- Keep a running "Secrets & Clues" list in each session — things the players may or may not discover
